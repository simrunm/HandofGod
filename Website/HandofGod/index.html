Our Project

Our goal in this project was to make an automated 2-dimensional gantry system which can catch a thrown ball based on camera recognition and prediction, with a focus on speed and accuracy

Project features

Real-Time Camera Tracking

Use of Integrated Web-Cam for Ball-Tracking for path prediction.

Arduino Microcontroller

Robust firmware for precise and fast stepper motor command

Stepper Motor Control

Precise movement and string control for accurate hand-motion.

String Tension Based Movement

String-based gantry for cheap and ingenious x-y movement.
